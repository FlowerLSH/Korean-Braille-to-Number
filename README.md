# Korean-Braille-to-Number

한국어 문장을 한국 점자 규칙에 맞게 분석하고, 각 점자 칸에서 찍어야 하는 점 번호를 출력하는 변환기입니다.

점자는 가로 2칸, 세로 3칸 구조입니다. 이 프로그램은 왼쪽 위부터 아래로 `1, 2, 3`, 오른쪽 위부터 아래로 `4, 5, 6`으로 보고, 채워야 하는 점을 숫자로 표시합니다.

```txt
1 4
2 5
3 6
```

예를 들어 `하늘이`는 다음처럼 출력됩니다.

```txt
245 14 2346 135
```

각 묶음은 점자 한 칸입니다. 입력의 빈칸은 점이 없는 칸이므로 `|`로 표시합니다.

## 주요 기능

- 한글 완성형 음절을 초성, 중성, 종성으로 분해
- 한국 점자 약자와 약어 판정
- 숫자, 로마자, 문장부호 토큰 판정
- 점자 칸별 점 번호 출력
- 브라우저에서 바로 쓰는 웹앱 제공
- CLI 출력 지원
- 분석 결과, 내부 ASCII, 순수 자모 분해, JSON 출력 지원

## 웹앱

[index.html](./index.html)을 브라우저에서 열면 바로 사용할 수 있습니다.

웹앱에서 제공하는 출력 모드는 다음과 같습니다.

- `점 번호`: 최종 점 번호 출력
- `분석`: 약자/약어/숫자/로마자/문장부호 판정 결과
- `ASCII`: 내부 점자 ASCII 표기
- `순수 분해`: 약자 판정 없는 초성/중성/종성 분해
- `JSON`: 구조화된 분석 결과

## CLI 사용

```bash
npm start -- "내돈내산"
```

출력:

```txt
14 1235 24 12356 14 1235 123 25
```

약자/약어 판정 결과를 보고 싶으면 `--analysis` 옵션을 사용합니다.

```bash
npm start -- --analysis "내돈내산"
```

출력:

```txt
ㄴㅐㄷ[약자:온]ㄴㅐ[약자:사]ㄴ
```

내부 점자 ASCII 표기는 `--ascii`로 볼 수 있습니다.

```bash
npm start -- --ascii "내돈내산"
```

출력:

```txt
cri(crl3
```

약자/약어 판정 없이 순수 초성, 중성, 종성만 보고 싶으면 `--plain` 옵션을 사용합니다.

```bash
npm start -- --plain "내돈내산"
```

출력:

```txt
ㄴㅐㄷㅗㄴㄴㅐㅅㅏㄴ
```

구조화된 분석 결과는 `--json`으로 확인할 수 있습니다.

```bash
npm start -- --json "산"
```

## 예시

```bash
npm start -- "내가 하늘에 서겠다."
```

```txt
14 1235 1246 | 245 14 2346 1345 | 6 234 4 1345 34 24 256
```

```bash
npm start -- "나는 Apple 3개를 샀다!"
```

```txt
14 14 1356 | 356 6 1 1234 1234 123 15 256 | 3456 14 4 1235 5 2346 | 123 34 24 235
```

```bash
npm start -- --analysis "꺾"
```

```txt
ㄲ[약자:억]ㄱ
```

표준 입력도 사용할 수 있습니다.

```bash
echo "내돈내산" | npm start
```

## 기준

한국어 점자 약자, 약어, 숫자, 로마자, 문장부호 처리는 국립국어원 `2024 개정 한국 점자 규정`을 기준으로 구현했습니다.

## 주의

이 도구는 국립국어원 한국 점자 규정을 참고해 구현한 보조 변환기입니다. 공식 점역 결과가 필요한 경우에는 최신 한국 점자 규정과 전문 검수를 함께 확인해야 합니다.
